# MyTube

A YouTube client application for iOS built with Swift.

## About

MyTube is an iOS application that provides a streamlined YouTube viewing experience. This project is designed to offer users an intuitive interface for browsing and watching YouTube content on their Apple devices.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/aristidesnakos/mytube.git
cd mytube
```

2. Open the project in Xcode:
```bash
open MyTube.xcodeproj
```
Or if using workspace:
```bash
open MyTube.xcworkspace
```

3. Build and run the project in Xcode (‚åò + R)

## Usage

After launching the app:
1. Browse the home feed for recommended videos
2. Use the search functionality to find specific content
3. Tap on any video to start playback
4. Enjoy your viewing experience!
